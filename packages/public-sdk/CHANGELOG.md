# Changelog

All notable SDK and public HTTP contract changes are documented here. Versions follow Semantic Versioning and track compatible `/public/v1` contract releases.

## 1.0.0 - 2026-07-11

- Generate TypeScript types from the canonical OpenAPI 3.1.1 contract.
- Add a typed fetch client restricted to `/public/v1` operations.
- Support scoped API key and OAuth2 access-token authentication without credential persistence.
